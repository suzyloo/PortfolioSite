Suzydiane was intended to be an all-encompassing portfolio site focused on my art, photography, and development endeavors. The design was supposed to be simple yet interesting, playing on my terrible sense of humor. The site is hosted through Heroku and runs with Python. Basic layout is completed with HTML/CSS with JS/JQuery used for animations and effects. Databases were created and maintained using SQLite 3, which is embedded within the site itself. The Python framework Flask was used. The site responds to page size.
